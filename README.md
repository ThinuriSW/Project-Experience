## Workplace Stress Simulation with Machine Learning and Reinforcement Learning

This project models workplace stress, burnout, absenteeism, and productivity using an agent-based simulation framework. It combines a rule-based model with machine learning and reinforcement learning to study coping behavior and organizational interventions under different workplace conditions.

[Explore Project](https://github.com/ThinuriSW/Project-Experience/tree/main/workplace_stress_abm_ml_rl)

## Occupational AI Vulnerability

This project examines how occupational skill composition shapes vulnerability to AI-driven change. Using clustering, principal component analysis, and a psychosocial buffer index, it studies how autonomy, social interaction, and task variety may protect workers from occupational stress and disruption.

[Explore Project](https://github.com/ThinuriSW/Project-Experience/tree/main/occupational_ai_vulnerability)
